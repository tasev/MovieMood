# MovieMood

MovieMood is an Android application that allows users to browse trending movies, view movie details, read reviews, and search for movies. The app is built using Kotlin, Jetpack Compose, and follows modern Android development practices.

## Features
- Browse trending movies
- View movie details
- Read movie reviews
- Search for movies

## Tech Stack
- **Kotlin**: Programming language
- **Jetpack Compose**: UI framework
- **Hilt**: Dependency injection
- **Retrofit**: Networking
- **Gson**: JSON parsing
- **Coil**: Image loading
- **ProGuard**: Code obfuscation and shrinking

## Project Structure
- `data`: Contains API interfaces, DTOs, and repository implementations.
- `domain`: Contains domain models and repository interfaces.
- `ui`: Contains UI components and screens.
- `di`: Dependency injection modules.
